# Review Delta Check Example

Use this example before finishing de-slop, greenfield public pages, reference-led redesigns, or visual quality audits.

## Required final check

Answer these in working notes before final delivery:

```text
Generic default avoided:
Product-specific proof shown:
Structural fingerprint:
Reference traits transformed:
Fabricated claims introduced: none | list and remove
Mobile recomposition review:
Checks run:
Remaining risk:
```

## Example

```text
Generic default avoided: centered gradient hero plus three feature cards.
Product-specific proof shown: install command and before/after UI review artifact.
Structural fingerprint: command bar, annotated comparison, compact mono metadata, restrained blue/red logo accent.
Reference traits transformed: high contrast and proof-led hero retained; original nav geometry and hero silhouette changed.
Fabricated claims introduced: none.
Mobile recomposition review: command wraps inside stable shell; comparison becomes vertical with preserved captions.
Checks run: build, typecheck, static UI review, manual desktop/mobile layout review.
Remaining risk: example output is illustrative until replaced by real user-submitted cases.
```
